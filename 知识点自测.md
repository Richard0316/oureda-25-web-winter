# 知识点自测

都是前端三件套比较基本和常用的一些知识点，建议食用方法：**先自己填一遍，每个问题再单独查一遍**。

*此部分不建议一上来就使用 AI，最终要求完成即可，重在自我检验。*
*本文为了可读性在代码处采用了 markdown 语法，能理解题意就行。*

## HTML

- Q1：你能说出哪些标签？与表单相关的是哪几个？
  A1：<div>, <p>, <span>, <h1>~ <h6>, <a>, <img>, <ul>, <ol>, <li>, <table>, <form>, <input>, <button>, <label>

  表单<form>, <input>, <textarea>, <select>, <option>, <button>, <label>, <fieldset>, <legend>

- Q2：补全 `style` 属性，使`<div style=""></div>`的宽为 `100px`、高为 `100px`。
  A2: `<div style="width:100px;height:100px;"></div>`

## CSS

- Q1：选择器有哪几种分类？
  A1：基础（其中有标签，类，ID，通配符）；组合（后代，子代，相邻兄弟和通用兄弟）；伪类和伪元素

- Q2：分别说几种伪类和伪元素？
  A2:伪类:hover;:active;:focus
伪元素:before;:after
- Q3：flex 布局和 grid 布局分别有哪些常用属性？
  A3：flex布局：
  display:flex;  flex-direction;
  justify-content;
  allgn-items;
  grid布局：
display:grid;
grid-template-columns;
gap;
justify-items;
grid-column-start;

- Q4：说说盒模型的构成？
  A4：内容区，内边距，边框，外边框

## JavaScript

- Q1：JS 中有哪些**基本数据类型**？
  A1：string,number,bigint,boolean,undufined,symbol,null

- Q2：如何通过 dom 操作获取 `id` 为 `test` 的元素，为其添加一个 "click" 事件？
  A2：`document.getElementByid('test').addEventListener`('click',function(){});

- Q3：箭头函数对比普通函数的区别有哪些？
  A3：语法更简单，但不能使用new关键字，没有arguments对象。

- Q4：已知一个变量 `num === 1`，写一个模版字符串，使其显示结果为 `“class1”`，即在 `"class"` 后面插入变量 `num`。
  A4：`<div>`class${num}`</div>`
